   ::::::::::::::::::    Scope of the test   ::::::::::::::::::
            
This test will fetch the given URLS from the excel sheet and write the broken links 
if any on the page against the urls.
To reuse the sheet please ensure that previous rows having some urls are deleted. 


Steps to use this project:

Pre-requisites:-
(1) Install JAVA (JDK 1.8 or higher version) on your machine.
(2) Download/Install eclipse


Execution Steps:

(3) Import project to eclipse and place urls in first column of IOfiles > Pfizer Link Test Results.xls 
to get tested for broken links.
Note: Excel sheet must be closed to get results.
(4) Run the test and wait for execution to complete.
